# My Minecraft Launcher

Personal Minecraft launcher supporting multiple game versions.

## Project Status
Currently in development.

## Purpose
This is a non-commercial personal project created for educational purposes.
It integrates with the official CurseForge API to retrieve metadata and authorized download links only.

## Compliance
- No file redistribution
- No external mirrors
- No monetization
- API usage in accordance with CurseForge TOS
